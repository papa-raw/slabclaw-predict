# Cardmarket source PDFs — the 4 hackathon cards

Print-to-PDF captures of each card's Cardmarket "Singles" offers page. The grade for a graded
slab is in the seller's **comment/note** (collapsed in print), not the MT/NM/EX condition badge
— that's the gotcha that made these the source of truth when the live scrape was blocked.

- `flareon-jungle-3.pdf` — base2-1st-3
- `dark-raichu-team-rocket-83.pdf` — base5-1st-83
- `typhlosion-neo-genesis-18.pdf` — neo1-1st-18
- `karens-umbreon-vs-091.pdf` — jp-vs-091

Used to validate `cardmarket-tinyfish.mjs` / `scan-cardmarket.mjs` grade extraction.
