Why This Isn't "Just Blob Storage"

Walrus provides the persistence layer for agent intelligence. This isn't about storing files — it's about agents that remember, learn, and coordinate over time.

🧠

Persistent Memory (MemWal)

Each agent maintains per-card observation history, anomaly logs, and reliability self-assessment. Memory persists across sessions — kill the process, restart, and agents pick up where they left off.

🤝

Shared Multi-Agent Context

9 agents coordinate through shared MemWal namespaces. The coordinator reads all signals. Each agent reads its own reputation score. No central database — just shared persistent memory.

🔍

Verifiable Evidence

Every consensus round produces an evidence blob on Walrus containing all inputs, weights, rejections, and the aggregation math. Anyone can fetch the blob and re-run the computation.

📈

Observable Learning

Source reliability diverges over rounds — eBay earns 96% while thin markets drop to 49%. Confidence intervals narrow from ±25% to ±4%. The swarm visibly gets smarter.

Manipulation Resistance

AttackStrategyCostDefenseOutcome
Single-source spoof Wash trade on eBay ~$6K fees MAD rejection (1/9 sources) Caught
Multi-source coordination Simultaneous wash on 4+ platforms ~$23K+ CUSUM drift + new signals lack reputation Expensive & detectable
Agent compromise Inject false signal via code Variable Transparent aggregation on Walrus exposes it Provably detectable
Coordinator compromise Produce false consensus Key access Evidence blob = anyone verifies the math Provably fraudulent

Key insight: The parimutuel prediction market structure itself resists many vectors. Wash trading is impossible (buying always increases the pool). Flash loans are neutralized (capital locked until resolution). Oracle manipulation requires simultaneously manipulating multiple independent real-world marketplaces — far more expensive than the typical pool size.

Walrus Evidence Layer

Every consensus round is a verifiable artifact

After the coordinator aggregates, the full evidence bundle is uploaded to Walrus as a permanent blob. The bundle contains: all source agent signals, per-agent confidence and comp counts, MAD z-scores for every rejected outlier, the weighted median computation, source reliability weights, and the final consensus price.

Before disputing an oracle proposal, anyone can download the evidence blob and independently verify the math. This makes frivolous disputes nearly impossible — the evidence is public, deterministic, and re-computable.

Observable Learning

The swarm demonstrably improves over time. Three visible indicators:

Reliability Divergence

Round 1: all sources weight 1.0. Round 10: eBay earns 96%, thin markets drop to 49%. The swarm learns which sources to trust.

Confidence Narrowing

Round 1: ±25% confidence interval. Round 10: ±4% interval. More data = tighter consensus = higher-quality oracle.

Anomaly Memory

Round 5: manipulation detected (fake 4x price). Round 6+: that pattern is pre-filtered. The swarm remembers attacks.
